# クイックへぇボタン

どこにでも埋め込めるへぇボタンです。[実際に試す](https://zhavtom.github.io/QuickHeeButton/)



# 必要環境

iframeが使えるブラウザ (2020年6月現在)



# 導入方法

```html
<iframe src="https://zhavtom.github.io/QuickHeeButton/"></iframe>
```

↑のコードを埋め込むだけです。必要に応じて`width`や`height`を設定してください。
